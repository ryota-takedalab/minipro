# mini project

「フィギュアスケートのジャンプにおけるエッジエラー判定の自動化」をテーマにミニプロに取り組んでいる。  
対象のジャンプは1Lz(シングルルッツ)。用いたデータセットやソースコードをここに公開する。

データ収集にはNOITOM社製慣性センサ"perceotion neuron 3.0"を用いた。
